Selecionando elementos com DOM

Depois de conhcermos a estrutura DOM, podemos selecionar os elementos que queremos manipular. Para isso há 5 maneiras diferentes. São elas: por Marca,por ID, por Nome, por Classe e por Seletor. Vejamos cada uma dessas e o resultado ontido

Por marca - "getElementByTagName( )"

Para selecionar todos os elementos de uma determinada tag, usamos o script getElementsByTagName( ). Abaixo está o script que criamos para selecionar um elemento pela marca(tag)

TEXTO EXEMPLO

                var p1 =window.document.getElementsByTagName('p')[2]
               window.document.write( 'Aqui está o texto selecionado >> ' + p1.innerText)
               p1.style.color = 'blue'

Veja que, para facilitar a selecão e posterior manipulação do elemento, criamos uma variável que chamamos de p1, dai atribuímos o elemento a esta variável. Isso possibilitou inclusive estlizar o elemento deixando-o na cor azul.

É também importante se atentar ao fato de que quandp a seleção e feita por marca (getElementsByTagName) precisamos determinar qual elemento visto que esta seleção incluin todos os elementos de tag similar . Por isso ao final da primeira linha do script depois de determinar qual a tag que estamos selecionando, precisamos definir qual elemento, é o que pode ser visto no colchete no final da primeira linha. Visto que em Javascript a contagem se inicia em zero e o texto exemplo estava na terceira tag de paragráfo o valor na chave foi [2].

Por ID - "getElementById( )"

Se um elemento tem um atributo id exclusivo, você pode selecioná-lo diretamente usando getElementById. Isso é eficiente e direto. Para isso precisamos definir um id para o elemento que queremos manipular. Vamos seguir usando o mesmo exemplo porém vamos repeti-lo em outra tag para que não haja conflito de scripts. Abaixo está o script que criamos para selecionar um elemento pela Id.

TEXTO EXEMPLO 2

            var p2 = window.document.getElementById('t2')
            window.document.write('Aqui está o texto selecionado '+ p2.innerText)
            t2.style.background = 'green'

Conforme pode ser visto, neste caso não precismos usar os colchetes [] visto que o elemento foi selecionado por uma Id exclusiva. Mas assim como no caso anterior também criamos uma variável para selecionar e manipular este elemento.

Algo interessante a destacar sobre a seleção do conteúdo de um elemento, é que quando usamos o comando "window.document.write( )" podemos escolher se queremos exibir apenas o texto "window.document.write(t2.innerText)" ou o texto com a formatação do HTMl "window.document.write(t2.innerHTMLt)". Vamos reescrever novamente esse script, mas dessa vez usando o comando "window.document.write(t2.innerHTML)" .

                var p2 = window.document.getElementById('t2')
                window.document.write ('Aqui está o texto selcionado '+ p2.innerHTML)

Veja que neste caso, o texto foi exibido com a formatação original, incluido o negrito da fonte.

Por nome - "getElementByName"

Como já fica bem evidente pelo subtítulo, este método seleciona o elemento pelo seu "name" que tambem deve ser definido para o elemento desejado assim como no caso da seleção por ID. Vejamos isso no script abaixo

TEXTO EXEMPLO 3

            var d = window.document.getElementsByName('msg')[0]
            window.document.write('Aqui está o texto ' +d.innerText)
            d.style.backgroundColor= 'yellow'

Por classe -"getElementByClassName"

Quando queremos selecionar elemesntos que fazem parte de uma mesm classe usamos o getElementByClassName.. Confira o script abaixo.

TEXTO EXEMPLO 4

            var msg2 =window.document.getElementsByClassName('ex4') [0]
            window.document.write('Olha ele aqui novamente! '+msg2.innerText)

Por seletor - "getElementsByquerySelector( )" ou "getElementsByquerySelectorAll( )"

querySelector permite selecionar o primeiro elemento que corresponde a um seletor CSS. querySelectorAll retorna todos os elementos que correspondem ao seletor CSS especificado.

Quando selecionamos um elemnto através do querySelector podemos usar a sintaxe usada em CSS. Neste caso, podemos escolher selcionar por ID ou por Class. bastando definir isso no elemento.
