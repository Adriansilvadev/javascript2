iniciando Estudos com DOM

o que é DOM?

O DOM (Document Object Model) é uma representação estruturada e hierárquica de documentos HTML, XML ou XHTML, que fornece uma interface programática para interagir e manipular o conteúdo, estrutura e estilo de documentos web. Em termos mais simples, o DOM é uma árvore de objetos que representa a estrutura de uma página web, e ele pode ser modificado dinamicamente por meio de scripts (como JavaScript).

Nesta página vamos considerar como o DOM pode ser usado para interagir e ou manipular os diversos elementos que compoêm uma página web.

Arvore DOM

A árvore DOM (Document Object Model) é uma representação hierárquica e estruturada de um documento HTML ou XML, organizando seus elementos em uma árvore de nós (nodes). Cada elemento, atributo, texto e outros componentes do documento são representados como nós na árvore, formando uma estrutura que reflete a organização do documento. Vamos explorar alguns conceitos relacionados à árvore DOM:

Nó Raiz

Nó raiz é o ponto de partida da árvore DOM e representa todo o documento. Em HTML, esse nó geralmente corresponde ao elemento html.

Nós Elemento

Cada tag HTML (por exemplo, div, p, h1) é representada por um nó de elemento na árvore DOM. Esses nós podem conter outros nós, formando a hierarquia da árvore.

Nós Texto

Os textos dentro de elementos HTML (o conteúdo entre as tags) são representados como nós de texto na árvore DOM.

Nós Atributo

Os atributos de elementos HTML são representados como nós de atributo na árvore DOM.

Relações Pai, Filho e Irmão

A relação entre nós na árvore é expressa através de conceitos como pai, filho e irmão.
>>Um nó pai contém nós filhos.
>>Um nó filho tem um nó pai.
>>Nós irmãos compartilham o mesmo pai.
