Formatação de Strings

Nesta página consideraremos a formatação de strings, que é o processo de manipulação da aparência de uma string para atender a determinados requisitos de apresentação. Em JavaScript, existem várias maneiras de formatar strings, permitindo a incorporação de variáveis, controle de espaçamento, e a definição de estilos específicos. Aqui estão algumas abordagens comuns:

Concatenação de Strings:

Esta é uma forma básica de formatar strings usando o operador de concatenação (+). Foi esse o método usado nos exemplos da página anterior. No entanto, pode se tornar menos legível com strings longas. Por isso de mmodo geral podemos optar por usar ou método, o Template Strings

Template Strings

As template strings são delimitadas por crases ( ` ` ) e permitem a incorporação de variáveis diretamente na string usando ${ }. Isso torna o código mais legível e flexível. Aproveitamos o exemplo da página anterior para demonstrar o uso de Template Strings. Veja abaixo como ficou o código. 

                var n1 = Number(window.prompt('Digite um número:'))
                var n2 = Number(window.prompt('Digite outro número:'))
                var s = n1 + n2
                window.alert(`A soma entre ${n1} e ${ n2}  é igual a ${s} ` )

Note como essa técnica torna mais facil a construção e a legibilidade do comando windo.alert
