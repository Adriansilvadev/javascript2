Tratamento de Dados

Na primeira página você inseriu seu nome, mas ele nào foi para lugar algum. Por isso nesta página fizemos o tratamento de dados, ou seja, usamos outro comando para exibir o dado coletado com o comando "window.prompt". Para isso, primeiro criamos uma váriável em que será armazenado o valor digitado no campo de nome da caixa de texto gerada pelo comando "window.prompt". Em seguida usamos o comando window.alert para exibir esta informação.

Abaixo temos o script necessário para isso. Veja que a váriável fica dentro do parentese, porem fora da aspas com um simbolo de + que chamamos de concatenação. Outra coisa é que quando precisamos adicionar outro elemento, tal como uma exclamação precisamos concatenar novamente. Fazemos isso por adicionar o simbolo de + e uma aspas. Consulte o Código fonte para uma melhor análise do código. 

            var nome = window.prompt("Qual seu nome?")
            window.alert('Seja bem-vindo, ' + nome +'! ')
