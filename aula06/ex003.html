Tratamento de Dados

Nesta página vimos o tratamento de dados numéricos. Usando parte dos comando da página anterior, fizemos algumas alteraçòes para obter 2 números do usuário com o objetivo de fazer a soma deles.

Abaixo temos o script necessário para isso. Veja que além de definir as 2 variáveis, também fizemos a conversão das 2 variáveis, visto que a linguagem JavaScript entende os valores digitados como sendo strings. Caso não fizessemos a conversão o resultado que apareceria na caixa do window.alert seria apenas uma concatenaçòa de n1 com n2. 

                var n1 = Number(window.prompt('Digite um número:'))
                var n2 = Number(window.prompt('Digite outro número:'))
                var s = n1 + n2
                window.alert('A soma dos números digitados é '+ s)

Essa é uma das maneiras de fazer a conversão de string para número, mas há outra maneira que está representada aqui em baixo. Veja que é necessário fazer um ajuste dependendo do tipo de número, se inteiro ( 1, 5,12 etc) ou real (1.2, 2.5 ,3.2 etc). Para inteiro usamos Number.parseInt. Para reais (fracionados) usamos Number.parseFloat.

            var n1 = Number.parseInt(window.prompt('Digite um número:'))
            var n2 = Number.parseInt(window.prompt('Digite outro número:'))
            var s = n1 + n2
            window.alert('A soma dos números digitados é '+ s)
        
            var n1 = Number.parseFloat(window.prompt('Digite um número:'))
            var n2 = Number.parseFloat(window.prompt('Digite outro número:'))
            var s = n1 + n2
            window.alert('A soma dos números digitados é '+ s)

Uma observação importante é que embora a conversão automática usando a sintaxe Number(n*), que usamos no primeiro exemplo seja muito prática, ela não elimina as demais, pois podem haver situações em que o resultado desejado seja para um número inteiro. Neste caso usa-se a sintaxe Number.parseInt(n*) . Já em outras o resultado desejado pode ser um número real (fracionado) neste caso deve se usar a sintaxe NumberparseFloat(n*).

Conversão de número para string

Podem ocorrer situações em que precisamos fazer uma conversão contraria, de string para número. Para isso ha 2 maneiras de fazer a conversão.

Strin(n)

O método String() converte explicitamente qualquer valor para uma string.

                var n1 = Number(window.prompt('Digite um número:'))
                var n2 = Number(window.prompt('Digite outro número:'))
                var s = n1 + n2
                window.alert('A soma dos números digitados é '+ String(s))

n.toString()

O método toString() é chamado em um valor numérico para convertê-lo em uma string.

                var n1 = Number(window.prompt('Digite um número:'))
                var n2 = Number(window.prompt('Digite outro número:'))
                var s = n1 + n2
                window.alert('A soma dos números digitados é '+ s.toString())
