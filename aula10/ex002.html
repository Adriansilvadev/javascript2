Neste caso, o resultado visual foi exatamente o mesmo. Porém, ao consultar o código fonte, notamos que dessa vez capturamos os eventos apartir do JavaScript. Examine o código e note que, quando capturados a partir do JavaScript os eventos são capturados com propriedades diferentes. Abaixo, veja como ficou o script neste caso. 

   var img = window.document.getElementById('ret')
        img.addEventListener('click', clicar)
        img.addEventListener('mouseenter', entrar)
        img.addEventListener('mouseout', sair)
      function clicar() {
          img.innerText = 'Clicou'
          img.style.backgroundColor = 'red'
      }
      function entrar() {  
         img.innerText = 'Entrou'
         img.style.backgroundColor ='yellow'
         img.style.color = 'black'
      }
      function sair() {
          img.innerText = 'Saiu'
          img.style.backgroundColor = ' rgba(106, 219, 31, 0.843)' 
      }
