Passe o Cursor

Conforme pode ser verificado no código fonte desta página, os eventos foram disparados diretamente do HTMl, ao lado das atribuições do elemento div que está sendo manipulado. Porém é possivel fazer isso através de outro método, usando os "EventListnner". veja isso na Página 2. Abaixo temos o script.

      
            var img = window.document.getElementById('ret')
          function clicar() {
              img.innerText = 'Clicou'
              img.style.backgroundColor = 'red'
          }
          function entrar() {  
             img.innerText = 'Entrou'
             img.style.backgroundColor ='yellow'
             img.style.color = 'black'
          }
          function sair() {
              img.innerText = 'Saiu'
              img.style.backgroundColor = ' rgba(106, 219, 31, 0.843)' 
          }
